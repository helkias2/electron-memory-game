How to Play

On clicking the Start button the game will populate a 3 x 12 grid cells randomly with numbers from 1 to 10 (or 1 to 5 on Easy mode). The numbers will disappear after few seconds. You have to click the X in a cell where each number was in ascending order, so first click the X where 1 was placed, then click the X where 2 was placed and so on till you are done with all the numbers.

Remember, you have to start with 1 and move on. Clicking a wrong cell will end the game.
